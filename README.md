# Custom ROI dengan OpenCV

Contoh script membuat Custom ROI di aplikasi OPENCV

### Cara Pakai
berikut shortcut yang digunakan :
q   : exit
r   : mulai select ROI
n   : buat roi baru

### Prerequisites

OpenCV 4.x
Python 3.x
